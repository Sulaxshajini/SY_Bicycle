## 📌 Overview

**SY Bicycle Management System** is a web-based application designed to streamline bicycle management operations. The system enables efficient handling of bicycles, customers, rentals, bookings, inventory, and administrative tasks through a user-friendly interface.

This project aims to improve operational efficiency, reduce manual work, and provide a seamless experience for both administrators and customers.

---

## ✨ Features

### 👤 User Features

* User Registration & Login
* Browse Available Bicycles
* Search and Filter Bicycles
* Bicycle Booking System
* Rental Management
* Customer Profile Management
* Booking History Tracking

### 🔧 Admin Features

* Secure Admin Dashboard
* Bicycle Management (Add, Edit, Delete)
* Customer Management
* Booking Management
* Rental Tracking

## 🚀 Installation Guide

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Sulaxshajini/SY_Bicycle.git
```

### 2️⃣ Move Project to XAMPP

Copy the project folder into:

```text
C:\xampp\htdocs\
```

### 3️⃣ Start Apache & MySQL

Open XAMPP Control Panel and start:

* Apache
* MySQL

### 4️⃣ Create Database

Open:

```text
http://localhost/phpmyadmin
```

Create a new database:

```sql
sy_bicycle
```

Import:

```text
database/sy_bicycle.sql
```

### 5️⃣ Configure Database Connection

Update your database credentials in:

```php
config.php
```

### 6️⃣ Run the Project

Open:

```text
http://localhost/SY_Bicycle
```
